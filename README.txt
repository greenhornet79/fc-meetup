== Description ==

Pull in data about Fort Collins meetup groups using the Meetup API

This plugin is mainly for teaching purposes for the Fort Collins WordPress meetup and is based on the [WordPress Plugin Boilerplate](http://wppb.io/).

A few notes about the sections above:

*   "Contributors" is a comma separated list of wp.org/wp-plugins.org usernames


== Installation ==


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('FC_Meetup_hook'); ?>` in your templates
